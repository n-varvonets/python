CloudFormation - Infrastructure as a code - создаем код, которьіе автоматизирует создание инфраструктурьі для кода

CloudFormation состоит из 2х вещей:
    - CloudFormation Stack : контейнер с созданньіми ресурсами, которьіе мьі зотим создать
    - CloudFormation Template : файл с програмньім кодом в котором мьі описуем какие ресурсьі создать(vpc,rds,ec2...) - шаблон

Формат синтаксиса кода(два разньіх структурньіх язьіков):
    - YAML  # комментьі можно оставлять
    - JSON

Шаблон инфраструктурьі запускаем в любом  AWS Region
