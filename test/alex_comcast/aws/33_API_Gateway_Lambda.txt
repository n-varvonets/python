API Gateway : позволяет создать "front door" для программ, что бьі получить доступ к ресурсам на aws.
    Т.е. доступ почтки ко всему что есть есть а акке aws черезе линк url
Lambda Func : взаимдествуем со всеми сервисасами в aws
API Gateway and Lambda Func together : если лямде дать разрешение то она сделает все in aws by get-request via url
