CDN - Content Delivery Network состоит из Edge Locations(неболшие датацентрьі для хранения кеша)
Єто полезно тогда, когда далекое расстрояние между коиентами и хостом с нашей прилажухой.
С помощью CDN можно разгрузить нагрузку на сервер, где много фронт работьі - сделать CloudFront

кеширование картинок из самого ближайшего сервера в edge location для ФРОНТА

Вопрос амазона : где использвать CDN или  Edge Locations? Ответ -  в CloudFront, т.е. єто для фронта - чисто кеширование картинок.
